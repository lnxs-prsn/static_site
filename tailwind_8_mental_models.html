1 — Spacing

🧠 Think of it like:

p-* = "bubble of air around the content inside the box"

m-* = "push other things away from the outside of the box"

gap-* = "breathing room between children — the spacing engine"

2 — Flexbox

🧠 Think of it like:

flex = "turn this box into a row of things"

flex-col = "stack them top to bottom instead"

justify-* = "where do things sit along the main direction"

items-* = "where do things sit across the other direction"

🧠 Think of it like:

5 — Color

🧠 Think of it like:

bg-blue-* = "paint the background of the box"

text-blue-* = "paint the letters"

border-blue-* = "paint the outline"

8 — Responsive & states

🧠 Think of it like:

md:flex-row = "switch to a row — but only on medium screens and wider"

hover:bg-blue-50 = "apply this only when the mouse is on top"

dark:bg-gray-800 = "apply this only in dark mode"

Tailwind is mobile-first — the plain class is the small screen. Prefixes layer on top as the screen grows.
