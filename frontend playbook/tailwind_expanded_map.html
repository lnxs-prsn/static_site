What lives inside each of the 8 Tailwind sections

1 — Spacing
padding sides
pt, pr, pb, pl — target one edge at a time
margin sides
mt, mr, mb, ml — push from one specific side
auto margin
mx-auto — center a block inside its parent
negative marginnew
-mt-4 — pull the element up, overlap things
space-betweennew
space-x-* / space-y-* — add margin between children without touching the parent
2 — Flexbox
direction
flex-row, flex-col and their reverse variants
wrappingnew
flex-wrap — when children don't fit, drop to next line instead of squishing
grow & shrinknew
flex-1, flex-none, flex-auto — how much a child stretches or holds its size
ordernew
order-first, order-last — reorder visually without changing the HTML
self-alignnew
self-start, self-end — override alignment for one specific child
justify & align
justify-*, items-*, place-* — where things sit on each axis
3 — Grid
columns
grid-cols-* — how many equal slots per row
spanning
col-span-*, row-span-* — stretch an item across slots
auto flownew
grid-flow-row/col — which direction new items get placed
custom sizingnew
grid-cols-[200px,1fr] — mix fixed and flexible slot widths
placementnew
col-start-2, col-end-4 — pin an item to a specific position
4 — Typography
size
text-xs → text-9xl — the font size dial
weight
font-thin → font-black — how heavy letters feel
line height
leading-* — breathing room between lines
letter spacing
tracking-* — breathing room between letters
alignmentnew
text-left, text-center, text-right, text-justify
decorationnew
underline, line-through, no-underline — marks on the text
overflownew
truncate, text-ellipsis, line-clamp-* — what happens when text doesn't fit
transformnew
uppercase, lowercase, capitalize — change case without editing HTML
5 — Color
background
bg-* — paint the box
text color
text-* — paint the letters
border color
border-* — paint the outline
opacitynew
bg-blue-500/50 — make any color semi-transparent with a slash
ring colornew
ring-* — a glowing outline used for focus states
divide colornew
divide-* — automatic border lines between children
6 — Sizing
fixed width
w-4, w-16 — always this exact size
relative width
w-1/2, w-full — based on the parent
min / maxnew
min-w-*, max-w-* — set a floor or ceiling on how wide it can get
height
h-*, min-h-*, max-h-* — same idea applied vertically
aspect rationew
aspect-video, aspect-square — keep width and height in proportion automatically
containernew
container mx-auto — a box that caps its own width and centers on the page
7 — Borders & Shadows
border width
border, border-2, border-t — thickness and which sides
border stylenew
border-solid, border-dashed, border-dotted — the line pattern
rounded corners
rounded-sm → rounded-full — how soft the corners feel
shadownew
shadow-sm → shadow-2xl — lift the element off the page visually
ringnew
ring-2, ring-offset-2 — outline outside the border, common for focus
outlinenew
outline-none, outline-2 — browser's native outline, usually removed on inputs
8 — Responsive & States
breakpoints
sm: md: lg: xl: — apply a class only above a certain screen width
hover / focus
hover: focus: — apply when the user interacts
dark mode
dark: — apply only when the OS is in dark mode
groupnew
group + group-hover: — when you hover the parent, style a child
peernew
peer + peer-checked: — when a sibling changes state, style another sibling
active / disablednew
active: disabled: — while pressing, or when the element is turned off
first / lastnew
first: last: — target only the first or last child in a list
